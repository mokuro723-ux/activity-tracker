APIキーの取り方（初回のみ・無料）
  1. Google Cloud Console を開く
  2. プロジェクトを新規作成
  3. 「APIとサービス」→「ライブラリ」で YouTube Data API v3 を有効化
  4. 「認証情報」→「認証情報を作成」→「APIキー」でキーを発行
  5. 発行された AIza... を上の欄に貼り付け
